Arduino toolbox

  • Arduino toolbox
    • ANALOG_READ_SB — Permet de réaliser l'acquisition d'une grandeur analogique sur le port Analogique d'une carte Arduino.
    • ANALOG_WRITE_SB — Permet d'envoyer une valeur continûment variable sur un port donné
    • ARDUINO_SETUP — Permet de configuration le port de communication série entre l'arduino et scilab.
    • DCMOTOR_SB — Permet de piloter un hacheur (relié à un moteur)
    • DIGITAL_READ_SB — Permet d'envoyer une valeur logique 0 ou 1 sur un port donné.
    • DIGITAL_WRITE_SB — Permet d'envoyer une valeur logique 0 ou 1 sur un port donné.
    • ENCODER_SB — Permet de compter un ensemble d'événements sur une ou plusieurs voies.
    • INTERRUPT_SB — Permet de compter un ensemble d'événements sur une ou plusieurs voies.
    • SERVO_WRITE_SB — Permet de piloter un servomoteur en renseignant l'angle de 0 à 180° souhaité
    • TIME_SAMPLE — Permet de spécifier la durée d'acquisition/pilotage et le pas d'échantillonnage
